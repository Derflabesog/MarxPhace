# MARX PHACE INVESTMENTS SALUTIONS Website

This is the official website for MARX PHACE INVESTMENTS SALUTIONS, a professional civil engineering and construction firm in Namibia.

## Hosting on GitHub Pages

To host this site on GitHub:

1.  Initialize a Git repository in this folder.
2.  Commit all files (`index.html`, `assets/`, etc.).
3.  Create a new repository on GitHub.
4.  Push your local files to the GitHub repository.
5.  Go to **Settings** > **Pages** in your GitHub repository.
6.  Select the `main` branch as the source and save.

## Project Structure

*   `index.html`: Main landing page.
*   `assets/css/`: Stylesheets.
*   `assets/js/`: JavaScript files.
*   `assets/images/`: Images.
